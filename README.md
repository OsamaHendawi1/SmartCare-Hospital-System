🏥 SmartCare Hospital Management System

A console-based Hospital Management System developed in C++ that simulates core hospital operations such as patient management, appointment scheduling, billing, and record viewing.

🚀 Overview

This project is designed as a practical implementation of core programming concepts using C++. It demonstrates structured programming, basic object-oriented design, and data handling using STL containers.

✨ Features

- 👤 Patient Management
  
  - Add new patients
  - View patient list
  - Delete patients

- 📅 Appointment Management
  
  - Book appointments
  - View all appointments

- 💳 Billing System
  
  - Create bills for services (Checkup / Surgery)
  - Automatic discount for high-cost services
  - Tax calculation

- 📊 Records Viewer
  
  - View all patients
  - View appointments
  - View billing records

- 🔐 Login System
  
  - Basic authentication with limited attempts

🛠️ Technologies Used

- C++
- STL (Vectors, Strings)
- Console I/O

▶️ How to Run

1. Clone the repository:

git clone https://github.com/OsamaHendawi1/SmartCare-Hospital-System.git

2. Open the project in Visual Studio or any C++ IDE

3. Compile and run the program

📸 Sample Flow

- Login → Main Menu
- Choose module (Patients / Appointments / Billing / Records)
- Perform operations interactively

⚠️ Notes

- Data is stored in memory (no file/database persistence yet)
- Login credentials are hardcoded for demonstration purposes
